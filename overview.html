dark analyst ops

파일로 넘기고, 스키마로 검증하고, 체크포인트로 이어간다.

`stock-agents`는 TradingAgents식 다중 역할 흐름을 직접 LLM API 호출 대신 Hermes/Codex CLI 파일 핸드오프로 실행한다. 현재 브랜치의 확정 계약은 shallow pipeline, bounded repair, manifest-safe resume이다.

stock-agents analyze SPY --date 2026-01-15 --runner mock --analysts market,news --debate-rounds 1 --risk-rounds 1 --depth shallow
8Typer commands: doctor, collect, build-tasks, run-task, validate, show-run, analyze, resume
3runner paths: mock, hermes, codex
1..3debate and risk round bounds
shallowonly implemented depth

Pipeline board

기본값은 market/news만 선택한다. `--analysts all`은 sentiment와 fundamentals를 추가한다.

01Selected analysts write evidence-grounded `AnalystOutput` files from inputs.market, news by default
02Bull and bear researchers debate validated analyst outputs. Multi-round runs write `_roundN` ids.--debate-rounds 1..3
03Research manager resolves the final bull/bear round into a balanced recommendation.05_research_manager
04Trader converts research into a cautious proposal without guaranteed returns.06_trader
05Aggressive, conservative, and neutral risk analysts review the proposal. Multi-round runs use final round dependencies.--risk-rounds 1..3
06Portfolio manager emits the final decision and must set `not_financial_advice=true`.10_portfolio_manager

Contract panels

코드에서 확인한 현재 보장과 안전장치다.

mock

Deterministic local path

테스트와 회귀 확인용. 네트워크, 모델 로그인, API key 없이 JSON fixture를 만든다.

hermes

Hermes CLI handoff

`hermes chat -Q`로 prompt를 넘긴다. provider와 model은 옵션이며 source는 `stock-agents`로 기록된다.

codex

Codex exec handoff

`codex exec --sandbox workspace-write`를 사용하고 `logs/codex/`에 stdout JSONL, stderr, last message를 저장한다.

manifest

Graph shape locked

`analyst_roles`, `debate_rounds`, `risk_rounds`는 manifest에서 재사용한다.

reject

Mismatches fail early

명시 옵션이 manifest와 다르면 새 task를 실행하기 전에 실패한다.

legacy

Old risk rounds default

기존 manifest에 `risk_rounds`가 없으면 1로 해석한다.

pydantic

Role schemas

Analyst, Debate, ResearchPlan, TraderProposal, RiskArgument, PortfolioDecision 출력 모델이 있다.

repair

One bounded repair

JSON 추출 또는 스키마 검증 실패 시 repair task를 한 번 만들고 재시도한다.

advice guard

Disclaimer required

Portfolio decision은 `not_financial_advice=true`를 요구한다.

Limits that matter

문서에서 과장하면 안 되는 현재 경계다.

depth

Only shallow depth

`standard`와 `deep`은 enum 이름으로만 존재한다. CLI pipeline은 shallow만 받는다.

data

Provider coverage

Market은 yfinance로 최소 1년 일봉을 수집한다. News는 yfinance를 먼저 쓰고, 한국 종목 뉴스가 없거나 부족할 때 네이버를 fallback으로 사용한다.

resume

No fact rebuild

Resume은 role outputs를 이어가지만 누락되거나 손상된 fact inputs를 다시 만들지는 않는다.
